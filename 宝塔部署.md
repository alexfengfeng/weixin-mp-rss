# 宝塔 Nginx 源码部署

## 环境

- Node.js LTS
- PM2 管理器
- Nginx
- SQLite
- Playwright 浏览器依赖

## 部署命令

```bash
cd /www/wwwroot/weixin-mp-rss
cp .env.example .env.production
npm install
mkdir -p data data/runtime data/cache/rss
touch data/app.db
npx prisma generate
npx prisma db push
npm run init:admin
npm run build
pm2 start ecosystem.config.cjs
pm2 save
```

## Nginx 反代

在宝塔站点配置中将请求反代到：

```text
http://127.0.0.1:3000
```

建议添加：

```nginx
client_max_body_size 20m;
proxy_read_timeout 300s;
proxy_send_timeout 300s;
proxy_set_header Host $host;
proxy_set_header X-Real-IP $remote_addr;
proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
proxy_set_header X-Forwarded-Proto $scheme;
```

## 旧数据迁移

```bash
npm run migrate:legacy -- --from=/www/wwwroot/we-mp-rss/data/db.db
```

token/cookie 默认不迁移，请在新后台重新扫码授权。

注意：Prisma SQLite 路径相对 `prisma/schema.prisma`，因此 `.env.production` 中的 `DATABASE_URL` 默认为 `file:../data/app.db`，实际数据库文件会在项目根目录的 `data/app.db`。
